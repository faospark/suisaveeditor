# File Structure

## Active Files
- `index.html` - Main editor page
- `test-gamedata.html` - Debug/test page for GameData
- `src/gamedata.js` - Game reference data (items, characters, runes, etc.)
- `src/main.js` - Editor logic and UI rendering
- `src/style.css` - Custom styles
- `css/pico.css` - Pico CSS framework
- `public/favicon.svg` - Site icon

- `src/javascript.svg` - Vite template leftover, not used
